# syz-logparser

`syz-logparser` is a small CLI that parses Linux kernel logs using syzkaller’s built‑in crash reporter. It extracts crash reports, prints them in a human‑readable form by default, or emits JSON for scripting.

## Building

```
./recompile.sh
```

The binary will be placed at `bin/syz-logparser`.

## Usage

```
bin/syz-logparser [flags] <kernel_log_file>
```

Flags:
- `-os` / `-arch` — target OS/arch of the log (default: current host).
- `-config` — optional syz-manager config to reuse parsing settings.
- `-json` — output parsed crashes as JSON.

Examples:
- Human-readable output: `bin/syz-logparser /path/to/kernel.log`
- JSON output: `bin/syz-logparser -json /path/to/kernel.log`

## Status

Tested only on amd64 with no config provided, both with and without `-json`. Other configurations/architectures are untested.
